# AutoHotkey 快捷键增强脚本

这是一个基于 AutoHotkey v2.0 的 Windows 快捷键增强脚本，提供了丰富的功能来提高您的工作效率。

## 功能概述

- **全局快捷键**：提供常用操作的快捷键映射
- **应用程序快速切换**：一键切换到常用应用
- **窗口置顶与半透明**：让窗口保持在最前面并调整透明度
- **任务栏控制**：快速显示任务栏
- **模拟键盘输入**：以不同速度模拟键盘输入剪贴板内容
- **连点器**：记录并重复执行鼠标点击操作
- **智能主题切换**：支持手动和基于日出日落的自动主题切换
- **应用程序特定快捷键**：为不同应用提供定制快捷键

## 安装与使用

1. 安装 [AutoHotkey v2.0](https://www.autohotkey.com/)
2. 下载本脚本
3. 双击运行 `hotkey.ahk` 文件（需要管理员权限）
4. 脚本将在后台运行，您可以通过系统托盘图标访问

## 特色功能详解

### 智能主题切换

脚本支持基于日出日落时间的自动主题切换功能：

- **自动模式**：根据当地日出日落时间自动在亮色和暗色主题间切换
- **手动模式**：可随时手动切换主题并关闭自动功能
- **信息显示**：可查看当前日出日落时间信息

### Alt+Alt 双击功能

新增的 Alt+Alt 双击功能可以快速发送 Ctrl+Alt+` 组合键，适用于各种需要该组合键的场景。该功能采用智能检测，避免长按 Alt 键时的误触发。

## 自定义

您可以根据自己的需求修改脚本文件 `hotkey.ahk`，添加或调整快捷键。脚本采用模块化设计，各功能相对独立，便于定制。

## 注意事项

- 本脚本需要管理员权限才能正常运行
- 某些应用程序可能会阻止快捷键工作
- 连点器功能应谨慎使用，避免在不适当的场景下使用
- 窗口半透明穿透功能会使窗口无法接收鼠标点击，需要再次使用 Alt+T 恢复正常
- 自动主题切换功能需要网络连接来获取日出日落时间
- Alt+Alt 双击功能的触发间隔为 300 毫秒，可根据个人习惯调整
